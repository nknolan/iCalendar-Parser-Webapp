## Assignment 3
03/2019 — iCalendar webapp.

### To Run

First run `npm install`, then go to the /parser directory and run `make`

To host the website at localhost, use `nmp run dev PORTNUM` where `PORTNUM` is whichever port number you want to use. This hosts the site at `localhost:PORTNUM`.

***

Has functions for uploading and downloading iCalendar files, reading existing files, updating files, and creating new files via GUI.
